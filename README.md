This project analyzes sales performance for an online e-commerce business using SQL Server. It uses a 5-table relational database (customers, products, categories, orders, order_items) with 60+ realistic orders to answer real business questions — top-selling products, monthly revenue trends, customer segmentation, and category performance. The project demonstrates SQL skills ranging from basic SELECT and GROUP BY queries to advanced window functions, CTEs, and subqueries, along with the ability to translate query results into business insights.
